# 🚀 Contributing to cForge

Thank you for your interest in contributing to `cForge`! This project aims to extend standard C functionality with **powerful string, memory, file, process, system, and networking utilities**.

We welcome all contributions, whether it's **bug fixes, new features, documentation improvements, or discussions**. Follow this guide to get started.

---

## 📌 1️⃣ Getting Started
### 🔽 **Fork & Clone the Repository**
1. **Fork this repository** by clicking the "Fork" button at the top of the repo.
2. **Clone your fork** to your local machine:
   ```sh
   git clone https://github.com/YOUR-USERNAME/cForge.git
   cd cForge
