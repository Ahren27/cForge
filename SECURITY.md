## Reporting a Vulnerability

1. **Use GitHub's Security Advisory Feature**  
   - Go to the cForge repository.  
   - Click on the "Security" tab → "Report a Vulnerability."  

2. **Submit an Issue (Private Option)**  
   - Open a GitHub issue and **mark it as security-related** (if private reporting is not available).  

We appreciate your help in keeping cForge secure! 🚀
